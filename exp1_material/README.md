# Items and Lists for RepeatitionMaze Experiment

The file *composed.csv* contains the originally composed versions of sentences and distracters. 


The R program *makeLists.R* checks the stimuli, create lists and adapt the stimuli for different uses.


The file *allitems.pdf* contains all items for easy visualization.

The file *material.json* contains all lists for jsPsych: there are four types of distracters (gramm, word, pseudo, nonword). For each type of distracters there are three counterbalanced lists (listA, listB, listC) in which **36 items** are presented in three condition (grammatical syntactic violation and semantic violation) in a latin square design.  Each list also contains the same three practice items (one for each conidtion).

The file *exp_lists.csv* contians assignment of items and conditions to different lists.

The file *material.json* contains all the lists to be used by the jspsych program which is found [../run](here).
